This file contains a short instruction in how to use our application effectively.

When first opening our application, you wil be faced with a main menu containing 3 options:
-Inventory
-Shopping List
-Account Settings

Inventory: This page will display all itmes currently stored in the syatem alng with the option to add or delete them.

Shopping List: This page will present the user with an option to add itmes to a shopping list as a tool to assist remembering which items are required.

Account sttings: Through this page the user wil be able to change their settings.
